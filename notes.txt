----------args-----------

select the required seed number.
    -1 or nothing will increment
    the previous seed by 1:       --seed <seed_number>
checkout specific branch:         --branch <branch_to_use>
run "make init":                  --init
run "make build-solution":        --build
pull the branch:                  --pull
make run-solution-extra-cpus:     --extra_cpus
make run-sim-no-gui               --no_sim_gui
run the sim with 3 min delay      --first_run_on_pre_init_and_build

e.g: python3 update_config.py --branch master --pull True --init True --build True --seed 30923

----------scripts-----------
python3 update_config.py --branch master --seed
OR
python3 update_config.py --branch master --pull True --seed
